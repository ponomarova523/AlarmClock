# AlarmClock 
Будильник на Java з графічним інтерфейсом (Swing).

## Функціонал
- Відображення поточного часу та дати
- Додавання будильників з вибором днів тижня
- Алгоритм сортування (Bubble Sort)
- Лінійний пошук будильників
- Функція snooze (відкладення)

## Запуск
```bash
javac AlarmClock.java
java AlarmClock
```

## Технології
Java 11+, javax.swing, javax.sound.sampled
